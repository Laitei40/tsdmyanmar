# Security Policy

## Supported Scope

TSD Myanmar is a static website hosted on Cloudflare Pages.
There is no backend server, database, or user authentication system.

Security concerns are limited to:
- Website integrity
- Dependency misuse
- Configuration issues
- Exposure of secrets or tokens

---
