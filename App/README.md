TSD Apps — Simple static showcase

Files:
- `index.html` — main page
- `style.css` — basic modern styles
- `script.js` — small script to render and filter apps

How to use:
- Open `index.html` in a browser (no build step needed).
- Edit `script.js` to add more apps or customize behaviour.

Design notes:
- Responsive grid, accessible modal, lightweight and easy to extend.
- Use icons or screenshots by replacing the inline icon building in `svgIcon`.
